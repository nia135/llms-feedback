
# LLM Feedback 
This repository contains the dataset used in our paper experiments for evaluating LLM-generated feedback.

## Dataset Structure

All dataset files are stored under the `dataset/` folder.

### 1. `dataset/train/`
Contains few-shot training examples used for prompting LLMs in the classification task.

### 2. `dataset/test/`
Contains two subsets of data used to evaluate the selected best LLM as an annotator:

- `Test_1.xlsx` – First evaluation dataset for calculating inter-annotator agreement and alternative test evaluations.  
- `Test_2.xlsx` – Second evaluation dataset for validating the LLM's performance.

### 3. `dataset/Feedback/`
Contains feedback generated by LLMs and annotated according to our taxonomy:

- `Gemini_Feedback.xlsx` – Feedback from Gemini 2.  
- `Llama 8B_Feedback.xlsx` – Feedback from LLaMA 8B.  
- `Mistral7B_Feedback.xlsx` – Feedback from Mistral 7B.  
- `Mistral24_Feedback.xlsx` – Feedback from Mistral 24B.  
- `Teacher_feedback.xlsx` – Feedback from human annotators.

## File Format

All files are in **Excel (.xlsx)** format.
